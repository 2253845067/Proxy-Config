# Proxy-Config

个人代理客户端配置模板与规则集整理仓库。

## 目录结构

- `mihomo/`: Mihomo 配置模板与片段。
- `mihomo/mihomo.yaml`: 当前使用的 Mihomo 主配置。
- `loon/`: Loon 配置文件。
- `surge/`: Surge 配置文件。
- `ruleset/`: 本地或生成后的规则集文件。

## 使用提醒

配置文件移动后，订阅链接里的路径也要同步更新。例如 Mihomo 主配置路径现在是：

```text
https://raw.githubusercontent.com/2253845067/Proxy-Config/main/mihomo/mihomo.yaml
```
